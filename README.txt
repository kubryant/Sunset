When was the last time you watch a sunset? Too long, most likely. Create a night sky with me and enjoy the Sunset. Control the bar at the bottom of the screen with the left and right keys on the keyboard. Press Esc to quit the game.

Repository Link: https://github.com/kubryant/Sunset/

Fork the repository to your own and build using Visual Studio 2010 or newer with XNA Game Studio Installed.
